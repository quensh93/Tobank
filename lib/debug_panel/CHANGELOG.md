# Changelog

All notable changes to the Debug Panel package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Debug Panel package
- Device preview functionality with multiple device models
- Logs viewer with filtering and search capabilities
- Accessibility testing and validation tools
- Responsive layout for desktop and mobile
- Tabbed interface for different debug tools
- Riverpod-based state management
- Comprehensive device models (iOS, Android, Desktop)
- Frame visibility and preview toggle controls
- Orientation switching support

### Features
- **Device Preview Tab**: Test apps on different device sizes and orientations
- **Logs Tab**: View and filter application logs with real-time updates
- **Accessibility Tab**: Comprehensive accessibility testing and validation
- **Responsive Design**: Adapts to different screen sizes
- **Extensible Architecture**: Easy to add new debug tools

### Technical Details
- Built with Flutter 3.10+
- Uses Riverpod for state management
- Follows Material Design 3 guidelines
- Comprehensive accessibility support
- Clean, modular architecture
