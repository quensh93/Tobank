# 
​
Center
The Stac Center allows you to build a Flutter center widget using JSON. To know more about the center widget in Flutter, refer to the official documentation.
## 
​
Properties
Property | Type | Description  
---|---|---  
widthFactor | `double?` | The factor by which to multiply the child’s width.  
heightFactor | `double?` | The factor by which to multiply the child’s height.  
child | `Map?` | The widget to display inside the center.  
## 
​
Example JSON
Copy
```
{
  "type": "center",
  "child": {
    "type": "text",
    "data": "Hello, World!"
  }
}

```
