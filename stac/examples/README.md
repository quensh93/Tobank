## File Naming Convention

- `widget_[name]_example.json` - Widget examples
- `action_[name]_example.json` - Action examples
- `complete_[feature]_example.json` - Complete UI examples

## Example Structure

```json
{
  "title": "Example Title",
  "description": "Brief description of what this example demonstrates",
  "json": {
    "type": "scaffold",
    "body": {
      "type": "custom_widget",
      "properties": {
        "example_property": "example_value"
      }
    }
  },
  "notes": [
    "Important implementation notes",
    "Configuration requirements",
    "Dependencies and prerequisites"
  ]
}
```
